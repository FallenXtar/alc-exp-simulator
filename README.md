# 众生之门游戏世界经验模拟

#### 介绍

众生之门游戏世界经验模拟

#### 软件架构

采用纯 Javascript 浏览器后端，前端使用 Vue + Element Plus 搭建。

#### 安装教程

1. 克隆仓库
2. 在根目录启动网页服务器
3. 访问 `index.html`

#### 使用说明

1.  在参数设置中填入模型的初始参数
2.  点击 `更新参数`
3.  点击 `初始化模型`
4.  运行模型
